In reply to: Boffosocko.com's Accessibility Statement

Posted by Robin Métral on 2022-04-10.

Boffosocko.com makes available the UserWay Website Accessibility Widget that is powered by a dedicated accessibility server. The software allows Boffosocko.com to improve its compliance with the Web Content Accessibility Guidelines (WCAG 2.0).

From Accessibility Statement, by Chris Aldrich.

Hi Chris! First of, thanks for the Indieweb inspiration on boffosocko.com!

Just a quick reaction here: I don't think you need to use UserWay to make your website accessible. At first glance I can't see any WCAG 2.0 violation here that the overlay would fix—but maybe I'm missing something?

In some cases, overlays can actually be harmful: I'm sure you've seen this already when everyone was sharing it a few months ago, but just in case you haven't, there's a lot of info on the Overlay Fact Sheet.

Happy to talk if I can help further with this, I'd love to "give back" somehow! Take care :)
